COLOR CODE

游戏说明

目标：用有限的模块将棋盘上所有关键点覆盖为对应颜色。

模块：每种模块的传色方式不同：

颜色传播：模块放置后会沿特定方向传播自身颜色，相邻模块可接收并混合颜色。

关键点：带虚线圆环的格子需要被对应颜色覆盖，全部匹配即可通关。

染色：被其他模块光线照射的模块会被染色——其颜色与入射光混合，并将新颜色继续向前传播，形成连锁反应。

羁绊：将特定模块相邻放置可触发协同效果：

操作方式：

从模块栏拖拽模块到棋盘
右键或双击旋转已放置的模块
Shift+点击移除模块

评级：完成速度决定评级 —— ▲ 三角（快）、■ 方块（中）、● 圆形（慢）。

资源：模块数量有限，请仔细规划摆放位置！

# COLOR CODE — 游戏规则

## 目标

用有限的模块将棋盘上所有**关键点**覆盖为对应颜色，全部匹配即可通关。

---

## 模块

共三种模块，传色方式各不相同：

- **△ 三角形** — 朝尖角方向无限射出颜色光线，点击可旋转方向（每次 90°）
- **■ 方块** — 向上下左右四个方向传导一格颜色，可作为路径桥接
- **● 圆形** — 向周围 3×3 范围内的所有格子辐射颜色

---

## 颜色传播与混合

模块放置后会沿特定方向传播自身颜色，相邻模块可接收并混合颜色。

被其他模块光线照射的模块会被**染色**——其颜色与入射光混合，并将新颜色继续向前传播，形成连锁反应。

| 混合 | 结果 |
|------|------|
| 红 + 蓝 | 紫 |
| 红 + 黄 | 橙 |
| 蓝 + 黄 | 绿 |

---

## 关键点

棋盘上带虚线圆环的格子为关键点，需要被**指定颜色**覆盖。全部匹配即通关。

---

## 羁绊

将特定模块相邻放置可触发协同效果：

- **▲ + ▲ ** — 光束扩展为 3 格宽的走廊
- **● + ■ ** — 方块获得对角线传播能力（8 方向）
- **▲ + ● ** — 圆形辐射范围扩展至 5×5，三角形获得侧面溅射
- **共鸣** — 同色但不同形状的模块相邻，传播距离延伸至 2 格

---

## 操作方式

| 操作 | 效果 |
|------|------|
| 拖拽模块到棋盘 | 放置模块 |
| 点击三角形 | 旋转方向（90°） |
| 拖动出棋盘格区域 | 移除模块 |
| 棋盘格之间拖动 | 移动已放置的模块 |
| 暂停按钮 | 暂停 / 继续计时 |
| 重置按钮 | 清空棋盘，归还所有模块 |

---

## 评级

完成速度决定评级：

- **▲ 三角** — 快
- **■ 方块** — 中
- **● 圆形** — 完成

---

## 资源限制

每关模块数量有限，请仔细规划摆放位置。
